# NAME

Acme::Shutup - shutup!

# SYNOPSIS

    > perl -MAcme::Shutup -e 'use Never::Exists::Module; use Whatever'

# DESCRIPTION

With Acme::Shutup, you'll never see `Can't locate XXX.pm in @INC` message.
How wonderful :)
